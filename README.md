# Конвертер EPUB сносок - PWA

Веб-приложение для преобразования концевых сносок в EPUB файлах в текст прямо в скобках.

https://typchik.github.io/epub-endnotes-to-parenthetical/ 

1. Загрузите EPUB файл (перетащите или выберите)
2. Приложение автоматически найдёт все сноски
3. Заменит ссылки на сноски текстом курсивом: `[1] [текст сноски]`
4. Скачает новый EPUB файл

## Особенности

- ✅ Работает полностью оффлайн (после первой загрузки)
- ✅ Все файлы обрабатываются локально в браузере
- ✅ Не требует подключения к интернету
- ✅ Сохраняет оригинальные ссылки
- ✅ Обрабатывает вложенные сноски
- ✅ Текст сносок выделен курсивом


## Файлы

- `index.html` - главный файл приложения
- `manifest.json` - манифест PWA
- `sw.js` - service worker для оффлайн работы
